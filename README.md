# Instructions

Alright guys, section's gonna be short and concise. Follow each step carefully if you don't wanna fuck up.

- Create an environment ```conda create -n knows python=3.10```
- Clone this repo. Then clone AgentLab-Knows inside this repo.
- pip install -r requirements.txt
- Install AgentLab using pip install -e . (cd to agent lab then run pip install -e .)
- Uninstall browsergym that comes with agentlab
- Install browsergym-core, experiments, webarena, knows in -e .
